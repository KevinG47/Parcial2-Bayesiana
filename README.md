Este repositorio contiene el desarrollo del Segundo Parcial de Estadística Bayesiana. Se han utilizado herramientas de R Markdown para garantizar que los resultados, las gráficas y las interpretaciones sean visibles directamente desde la plataforma sin necesidad de ejecutar el código localmente.

##  Guía de Revisión

Para ver los puntos resueltos con sus respectivas gráficas (posteriores, simulaciones y cadenas de Markov), por favor accede a los siguientes enlaces de los archivos **`.md`**:

* [**Punto 1: Decisión Bayesiana**](PrimerPunto_Bayesiana_KevinChaparro.md)
  * Modelos Beta-Binomial, Gamma-Poisson y Gamma-Exponencial.
  * Comparación de estimadores (media vs. mediana) e intervalos de credibilidad.

* [**Punto 7: Simulación Monte Carlo**](SeptimoPunto_Bayesiana_KevinChaparro.md)
  * Aproximación de esperanza y varianza para distribuciones Weibull y Gumbel.
  * Comparación de resultados simulados frente a valores teóricos.

* [**Punto 8: Algoritmo Metropolis-Hastings (MCMC)**](OctavoPunto_Bayesiana_KevinChaparro.md)
  * Implementación del algoritmo para una distribución normal.
  * Análisis de trayectoria, autocorrelación y discusión de convergencia.

##  Estructura del Repositorio

Para el correcto funcionamiento de la previsualización en GitHub, el repositorio mantiene la siguiente organización:

1. **Archivos `.md`**: Son los documentos principales de lectura que integran texto y resultados.
2. **Carpetas `_files/`**: Contienen las imágenes generadas por RStudio (gráficos de ggplot2 y base R). **Nota:** No mover ni renombrar estas carpetas para no romper los enlaces de las imágenes en los documentos de lectura.
3. **Archivos `.Rmd`**: Se adjunta el código fuente original como respaldo de la lógica y programación utilizada.
