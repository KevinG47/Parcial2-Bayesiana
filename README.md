# Segundo Parcial - Estadística Bayesiana

Este repositorio contiene los códigos en R correspondientes a los puntos prácticos del segundo parcial. 

## Archivos incluidos:

* `PrimerPunto_Bayesiana_KevinChaparro.R`: Código base para el modelo Beta-Binomial (análisis de la distribución posterior).
* `SeptimoPunto_Bayesiana_KevinChaparro.R`: Simulación con el método de Monte Carlo para las distribuciones Weibull y Gumbel, incluyendo la estimación de esperanza y varianza.
* `OctavoPunto_Bayesiana_KevinChaparro.R`: Implementación del algoritmo Metropolis-Hastings (MCMC) para una distribución normal, con la trayectoria de la cadena y autocorrelación.
